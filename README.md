# Web Project Homeland

Projeto final do Sprint 5 do Bootcamp Tripleten: Adaptive Web Design.

## Visualizar o projeto

Veja o projeto online no GitHub Pages:

👉 [https://yuriiloureiro.github.io/web_project_homeland/](https://yuriiloureiro.github.io/web_project_homeland/)

## Sobre

Este projeto foi desenvolvido seguindo a metodologia BEM Flat, com foco em responsividade para Desktop, Tablet e Mobile.

## Tecnologias usadas

- HTML5 semântico
- CSS3 com media queries
- Fonte Inter (weights 400 e 900)
- Git e GitHub Pages para versionamento e hospedagem
